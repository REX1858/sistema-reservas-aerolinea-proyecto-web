Actúa como un **diseñador senior UX/UI especializado en prototipos de aplicaciones web en Figma**.

Debes diseñar la **estructura completa de prototipos de una aplicación web de gestión de reservas de una aerolínea**, basada en el siguiente proyecto académico.

El objetivo es **definir todas las pantallas, navegación y flujo de usuario**, sin implementar código ni base de datos.

---

# CONTEXTO DEL PROYECTO

Una aerolínea nacional desea un sistema web que permita:

* Registrar clientes
* Realizar reservas de vuelos
* Comprar tiquetes
* Adquirir paquetes turísticos según el destino
* Administrar reservas y estados
* Gestionar vuelos y clientes

El sistema será desarrollado posteriormente con **HTML5, CSS (Flexbox o Bootstrap) y JavaScript bajo arquitectura MVC**, por lo que el diseño debe ser **claro, funcional y fácil de implementar**.

---

# PERFILES DE USUARIO DEL SISTEMA

El sistema debe considerar tres roles:

### Súper Administrador

Puede gestionar todo el sistema:

* vuelos
* destinos
* paquetes turísticos
* clientes
* parámetros del sistema

### Agente de Aerolínea

Encargado de:

* gestionar reservas
* confirmar pagos
* asignar asientos
* atender solicitudes de clientes

### Cliente

Puede:

* registrarse
* iniciar sesión
* buscar vuelos
* realizar reservas
* comprar tiquetes
* adquirir paquetes turísticos

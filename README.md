# sistema-reservas-aerolinea-proyecto-web
Proyecto integrador Ambiente Web I - Gestion de reservas y venta de servicios de aerolinea.

## Documentacion UML

### Modelo de Navegacion del Sitio Web

El esquema de diagramas UML se encuentra en:

- `docs/uml/modelo-navegacion/diagrama-navegacion-roles.puml`
- `docs/uml/modelo-navegacion/visual-diagramas.html` (version visual)

Incluye:

- Site map UML con menus por rol (Cliente, Agente y Super Admin).
- Enlaces condicionales por autenticacion y resolucion de rol.
- Flujo principal de reserva y compra (detalle -> reserva -> paquetes -> confirmacion).
